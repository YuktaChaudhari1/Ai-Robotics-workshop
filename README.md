## 📖 Project Overview

This project was developed as part of an assignment to create a workshop landing page inspired by the design style of Kidrove.

The website promotes an **AI & Robotics Summer Workshop** for children aged **8–14 years** and provides a registration form for submitting workshop enquiries.

---

## ✨ Features

* **Responsive Hero Section**
* **Workshop Details Section**
* **Learning Outcomes Section**
* **FAQ Section**
* **Registration Form**
* **Form Validation**
* **Express.js Backend API**
* **MongoDB Database Integration**
* **Mobile-Friendly Design**
* **Modern UI with Tailwind CSS**

## 🔗 API Endpoint

### POST `/api/enquiry`

Accepts the following data:

* **Name**
* **Email**
* **Phone Number**

### Functionality

* Validates required fields
* Validates email format
* Validates phone number
* Stores enquiry data in MongoDB
* Returns success/error responses
